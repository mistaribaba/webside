Unit  -  1  Introduction to HTML


1.1 Introduction to HTML
  • The full form of HTML is Hypertext Markup Language.
  • HTML is used to create the structure of a website.
  • HTML5 is the latest version of HTML.
  • The following is the history of HTML:
    • HTML 1.0 (1991)
    • HTML 2.0 (1995)
    • HTML 3.2 (1997)
    • HTML 4.01 (1999)
    • XHTML (2000)
    • HTML5 (2014)
  • Structure of HTML:
     <!DOCTYPE html> 

              <(name of the tag) (attribute)>(content)</(closing tag)>

    !!Remember!!
    Some tags do not have ending tags; they only have starting tags.

  • In HTML, there are tags and attributes for those tags.
  • HTML is not a case-sensitive language (except when using IDs and classes). Using uppercase in HTML is not a good practice.
1.2 Terminologies used in Web Design:
  • Web Page: A document on the World Wide Web that can be viewed in a web browser.
  • Website: A collection of related web pages hosted on a web server.
  • Web Browser: Software used to access and display web pages (e.g., Chrome, Firefox).
  • Web Server: A computer that stores and serves web pages to users.
  • URL (Uniform Resource Locator): The address of a web page.
  • Search Engines: Software used to store and provide information to users.
    Examples: Google, Yahoo!, Bing, etc.

1.3 Web page structure:
  • A web page is made up of two parts:
    • Front-end: The part of the web page that the user can see and interact with. Made up of HTML, CSS, and JavaScript.
    • Back-end: The part of the web page that the user cannot see. Made up of server-side languages like PHP, Python, Node.js, etc.
  • Basic structure of an HTML document: 
                                         
                  <!DOCTYPE html> 
                 <html lang="en"> 
                     <head> 
                         <meta charset="UTF-8"> 
                         <meta name="viewport" content="width=device-width, initial-scale=1.0">

                     <body>

  • <!DOCTYPE html>:
    • First line of code in the HTML document.
    • Informs the browser that this code uses HTML5.

    • The root element of the HTML document.
    • Contains all other HTML elements.

  • Paragraphs:
    • The <p> tag is used to define a paragraph in HTML.
    • Paragraphs are block-level elements and always start on a new line.
    • You can use multiple <p> tags to separate different blocks of text.
    • Browsers automatically add some space (margin) before and after each <p> element.
    • Syntax:
      <p>This is a paragraph.</p>
  • Breaks
    • The <br> tag is used to insert a line break within text or between elements.
    • <br> is an empty (self-closing) tag, meaning it does not have a closing tag.
    • It is commonly used inside paragraphs or headings to break lines without starting a new block element.
  • Divisions
    • The <div> tag is used to define a division or a section in an HTML document.
    • <div> is a block-level element and is commonly used as a container for other HTML elements to style them with CSS or to perform certain tasks with JavaScript.
    • It does not add any visual change by itself, but helps in grouping elements for layout and styling purposes.
  • Centered Text
    • You can center text in HTML using the text-align: center; CSS property.
    • The <center> tag was used in older HTML but is now obsolete; use CSS instead.
    • Example: <p style="text-align: center;">This text is centered.</p>
  • Preformatted text:
    • The <pre> tag displays text exactly as written, preserving spaces and line breaks.
    • Useful for showing code snippets or formatted text blocks.
    • Browsers render content inside <pre> in a fixed-width (monospace) font by default.
  • address:

    •  CSS Border Images
            
        
              
      • 
                What is a Border Image?
        
                Normally, when you add a border to an element (like a div or a button), you get a solid color, a dashed
                line, or something simple. But what if you want your border to look fancyâ€”like a frame, a pattern, or
                even a picture? That's where border images come in!
              
        • 
              
      • 
                How Does It Work?
        
                With CSS, you can use an image file (like PNG, JPG, SVG) as the border of your element. The browser
                "slices" the image and stretches or repeats it around the edges.
              
        • 
              
      • 
                Basic Syntax:
                
        
        /* Apply a border image to a box */
        .box {
          border: 10px solid transparent; /* You need a border for the image to show! */
          border-image: url('border.png') 30 round;
        }
                
        
                
          
                  
        • border: Sets the width and style (must not be "none" or "hidden").
          • 
                  
        • border-image: The magic property!
          
                    url('border.png') is your image.
          
                    30 is how much to "slice" from each edge (in pixels).
          
                    round tells the browser how to fit the image (other options: stretch, repeat).
                  
          • 
                
        
              
        • 
              
      • 
                Step-by-Step Example:
                
        
        /* HTML */
        <div class="fancy-border">Hello, Border Image!</div>

        /* CSS */
        .fancy-border {
          border: 12px solid transparent;
          border-image: url('https://css-tricks.com/examples/BorderImage/css3border.png') 27 repeat;
          padding: 20px;
          font-size: 1.2em;
        }
                
        
                
        
                  What happens here? The image is cut into 9 parts (like a tic-tac-toe grid), and the corners
                  stay in the corners, the sides stretch or repeat, and the center is ignored or filled.
                
        
              
        • 
              
      • 
                Why Use Border Images?
                
          
                  
        • Make your boxes, cards, or buttons look unique and creative.
          • 
                  
        • Add decorative frames without extra HTML elements.
          • 
                  
        • Great for fun projects, portfolios, or whenever you want to impress!
          • 
                
        
              
        • 
              
      • 
                Tips for Beginners:
                
          
                  
        • Start with a simple border image (like a pattern or a frame PNG).
          • 
                  
        • Use border-image-slice to control how much of the image is used for the border.
          • 
                  
        • Try different border-image-repeat values: stretch, repeat, round.
          • 
                  
        • Always set a border width and style, or the image won't show!

                What are CSS Text Effects?
        
                CSS text effects are special styles and decorations you can apply to text to make it stand out, look
                creative, or improve readability. These effects can include shadows, outlines, transformations,
                gradients, animations, and more.
              
        •

        • Make headings and important text eye-catching.
          • 
                  
        • Improve the look and feel of your website.
          • 
                  
        • Guide the user's attention to key information.
          • 
                  
        • Add fun or branding to your site.
          • 
                
        
              
        • 
              
      • 
                Common CSS Text Effects (with Syntax & Examples):
                
          
                  
        • 
                    1. Text Shadow
          
                    Adds a shadow behind your text for a 3D or glowing effect.
          
                    Syntax:
                    
          text-shadow: offset-x offset-y blur-radius color;
          
                    Example:
                    
          
      h1 {
        text-shadow: 2px 2px 8px #ff0000;
      }
                
          
                    Result: The heading will have a soft red shadow.
                  
          • 
                  
        • 
                    2. Text Outline (Stroke)
          
                    Outlines text with a colored border. Not standard in all browsers, but works in WebKit browsers.
          
                    Syntax:
                    
          
      h2 {
        -webkit-text-stroke: 2px black;
        color: white;
      }
                
          
                    Result: White text with a black outline.
                  
          • 
                  
        • 
                    3. Uppercase, Lowercase, Capitalize
          
                    Changes the case of your text.
          
                    Syntax:
                    
          
      p {
        text-transform: uppercase; /* or lowercase, capitalize */
      }
                
          
                    Example: All text in the paragraph will be uppercase.
                  
          • 
                  
        • 
                    4. Letter Spacing & Word Spacing
          
                    Adjusts the space between letters or words.
          
                    Syntax:
                    
          
      h3 {
        letter-spacing: 4px;
        word-spacing: 10px;
      }
                
          
                  
          • 
                  
        • 
                    5. Text Alignment
          
                    Aligns text left, right, center, or justify.
          
                    Syntax:
                    
          
      p {
        text-align: center;
      }
                
          
                  
          • 
                  
        • 
                    6. Text Decoration
          
                    Adds underline, overline, line-through, or removes decoration.
          
                    Syntax:
                    
          
      a {
        text-decoration: underline; /* or none, line-through, overline */
      }
                
          
                  
          • 
                  
        • 
                    7. Text Gradient (Advanced)
          
                    Makes your text display a gradient color.
          
                    Syntax:
                    
          
      h1 {
        background: linear-gradient(to right, #f00, #00f);
        -webkit-background-clip: text;
        -webkit-text-fill-color: transparent;
      }
                
          
                    Result: The heading text will have a red-to-blue gradient.
                  
          • 
                  
        • 
                    8. Animated Text Effects (Beginner Example)
          
                    You can animate text color, shadow, or other properties.
          
                    Syntax:
                    
          
      @keyframes colorChange {
        0% { color: red; }
        100% { color: blue; }
      }
      span {
        animation: colorChange 2s infinite alternate;
      }
                
          
                    Result: The text inside <span> will smoothly change from red to blue and

    • CSS 2D Transforms
            
        
              
      • 
                What are CSS 2D Transforms?
        
                CSS 2D Transforms let you move, rotate, scale, and skew elements on your web page in two dimensions (X
                and Y axes). This means you can make things slide, spin, grow, shrink, or tiltâ€”without using images or
                JavaScript!
              
        • 
              
      • 
                Why Use 2D Transforms?
                
          
                  
        • Make buttons or images move when hovered.
          • 
                  
        • Create fun animations and interactive effects.
          • 
                  
        • Improve user experience by making your site feel more dynamic.
          • 
                
        
              
        • 
              
      • 
                How to Use 2D Transforms?
        
                You use the transform property in CSS. Here are the main functions:
                
          
                  
        • translate(x, y): Moves the element left/right (X) and up/down (Y).
          • 
                  
        • rotate(angle): Rotates the element clockwise or counterclockwise.
          • 
                  
        • scale(x, y): Makes the element bigger or smaller.
          • 
                  
        • skew(x-angle, y-angle): Tilts the element along X and/or Y axes.
          • 
                  
        • matrix(...): Advanced, combines all above (not needed for beginners).
          • 
                
        
              
        • 
              
      • 
                Syntax & Examples:
                
          
                  
        • 
                    Move (Translate):
                    
          
      .box {
        transform: translate(50px, 20px); /* 50px right, 20px down */
      }
                  
          
                  
          • 
                  
        • 
                    Rotate:
                    
          
      .box {
        transform: rotate(45deg); /* Rotates 45 degrees clockwise */
      }
                  
          
                  
          • 
                  
        • 
                    Scale (Resize):
                    
          
      .box {
        transform: scale(1.5, 1.5); /* 1.5 times bigger in both directions */
      }
                  
          
                  
          • 
                  
        • 
                    Skew (Tilt):
                    
          
      .box {
        transform: skew(20deg, 10deg); /* Tilts 20deg on X, 10deg on Y */
      }
                  
          
                  
          • 
                  
        • 
                    Combine Multiple Transforms:
                    
          
      .box {
        transform: translate(30px, 0) rotate(-15deg) scale(1.2);
      }
                  
          
                  
          • 
                
        
              
        • 
              
      • 
                Beginner-Friendly Example:
        
                
        
      /* HTML */
      <div class="demo-box">Hover me!</div>

      /* CSS */
      .demo-box {
        width: 120px;
        height: 120px;
        background: #90fd4c;
        color: #222;
        display: flex;
        align-items: center;
        justify-content: center;
        font-weight: bold;
        transition: transform 0.3s;
        margin: 20px;
        border-radius: 12px;
      }
      .demo-box:hover {
        transform: scale(1.2) rotate(10deg) translateY(-10px);
      }
                
        
                What happens? When you hover over the box, it grows, tilts, and moves up a bit!
              
        • 
              
      • 
                Tips for Beginners:
                
          
                  
        • Always use transition for smooth effects.
          • 
                  
        • Try combining transforms for creative results.
          • 
                  
        • Use deg for angles, px or % for distances.

    • CSS 3D Transforms
            
        
              
      • 
                What are CSS 3D Transforms?
        
                CSS 3D Transforms let you move, rotate, scale, and skew elements in three dimensions (X, Y, and Z axes).
                This means you can make elements look like they're flipping, spinning, or popping out of the
                pageâ€”creating cool, interactive effects without JavaScript!
              
        • 
              
      • 
                Why Use 3D Transforms?
                
          
                  
        • Make cards, images, or buttons flip or rotate in 3D.
          • 
                  
        • Add depth and realism to your web designs.
          • 
                  
        • Create interactive UI effects that impress users.
          • 
                
        
              
        • 
              
      • 
                How to Use 3D Transforms?
        
                You use the transform property in CSS, just like with 2D transforms, but with extra
                functions for 3D:
                
          
                  
        • rotateX(angle): Rotates the element around the X-axis (horizontal flip).
          • 
                  
        • rotateY(angle): Rotates the element around the Y-axis (vertical flip).
          • 
                  
        • rotateZ(angle): Rotates the element around the Z-axis (like 2D rotate).
          • 
                  
        • translateZ(distance): Moves the element closer or farther from the viewer.
          • 
                  
        • scaleZ(factor): Scales the element in the Z direction.
          • 
                  
        • perspective(value): Adds depth so 3D effects look realistic.
          • 
                
        
              
        • 
              
      • 
                Syntax & Examples:
                
          
                  
        • 
                    Simple 3D Rotation (Y-axis):
                    
          
      .box3d {
        transform: rotateY(45deg);
      }
                  
          
                  
          • 
                  
        • 
                    Adding Perspective (for Real 3D Look):
                    
          
      .parent {
        perspective: 600px;
      }
      .box3d {
        transform: rotateX(30deg) rotateY(30deg);
      }
                  
          



                    What happens? When you hover over the card, it flips in 3D to show the back!
                  
          • 
                
        
              
        • 
              
      • 
                Tips for Beginners:
                
          
                  
        • Always use perspective on a parent for 3D effects.
          • 
                  
        • Use transform-style: preserve-3d; on the element you want to transform in 3D.
          • 
                  
        • Try rotateX and rotateY for flipping effects.
          • 
                  
        • Experiment with translateZ to move elements "closer" or "farther" away.
          • 
                
        
              
        • 
              
      •

    • CSS Transitions & Transforms 
            
        
              
      • 
                What are CSS Transitions?
        
                CSS transitions let you smoothly animate changes in CSS properties, such as color, size, position, or
                opacity, when something happens (like hovering or clicking). Instead of changes happening instantly,
                transitions make them gradual and visually appealing.
              
        • 
              
      • 
                Why Use Transitions?
                
          
                  
        • Make your website feel modern and interactive.
          • 
                  
        • Guide the user's attention to changes (like buttons, menus, or cards).
          • 
                  
        • Add polish and professionalism to your design.
          • 
                
        
              
        • 
              
      • 
                How to Use CSS Transitions?
        
                You use the transition property on the element you want to animate. You can specify which
                CSS property to animate, how long it should take, and the timing function (speed curve).
                
        
                Basic Syntax:
                
        
      /* Animate a property (e.g., background-color) over 0.5 seconds */
      .selector {
        transition: property duration timing-function delay;
      }

      /* Example: */
      .button {
        background-color: #90fd4c;
        transition: background-color 0.3s ease;
      }
      .button:hover {
        background-color: #222;
      }
                
        
                What happens? When you hover over the button, the background color smoothly changes instead of
                switching instantly.
              
        • 
              
      • 
                Commonly Animated Properties:
                
          
                  
        • color (text color)
          • 
                  
        • background-color
          • 
                  
        • border
          • 
                  
        • opacity (fade in/out)
          • 
                  
        • transform (move, scale, rotate, etc.)
          • 
                  
        • box-shadow
          • 
                
        
              
        • 
              
      • 
                Transition Timing Functions:
                
          
                  
        • ease (default): Starts slow, speeds up, then slows down.
          • 
                  
        • linear: Same speed from start to finish.
          • 
                  
        • ease-in: Starts slow, then speeds up.
          • 
                  
        • ease-out: Starts fast, then slows down.
          • 
                  
        • ease-in-out: Slow at start and end.
          • 
                
        
              
        • 
              
      • 
                Multiple Properties:
        
                You can animate several properties at once:
                
        
      .box {
        transition: background 0.3s, transform 0.5s;
      }
                
        
              
        • 
              
      • 
                Beginner-Friendly Example: Button Grow on Hover
                
        
      /* HTML */
      <button class="grow-btn">Hover Me!</button>

      /* CSS */
      .grow-btn {
        background: #90fd4c;
        color: #222;
        border: none;
        padding: 16px 32px;
        font-size: 1.2em;
        border-radius: 8px;

                What happens? When you hover, the button smoothly grows and changes color.

    • CSS Animations
            
        
              
      • 
                What are CSS Animations?
        
                CSS Animations allow you to create complex, smooth, and visually appealing animations directly in your
                CSSâ€”no JavaScript needed! You can animate almost any CSS property (color, size, position, opacity, etc.)
                and control how and when the animation happens.
              
        • 
              
      • 
                Why Use CSS Animations?
                
          
                  
        • Bring your website to life with movement and transitions.
          • 
                  
        • Guide user attention to important elements (like buttons, alerts, or banners).
          • 
                  
        • Make your site feel modern, interactive, and fun!
          • 
                
        
              
        • 
              
      • 
                How Do CSS Animations Work?
        
                There are two main parts:
                
          
                  
        1. @keyframes: Define the steps of your animation (what changes, and when).
          2. 
                  
        2. animation properties: Attach the animation to an element and control how it runs (duration,
                    timing, repeat, etc.).
          3. 
                
        
              
        • 
              
      • 
                Beginner Example: Fading In a Box
                
        
        /* 1. Define the animation steps */
        @keyframes fadeIn {
          from { opacity: 0; }
          to   { opacity: 1; }
        }

        /* 2. Attach the animation to an element */
        .box {
          opacity: 0;
          animation: fadeIn 2s ease-in forwards;
        }
                
        
                What happens? The box starts invisible and smoothly fades in over 2 seconds.
              
        • 
              
      • 
                Common Animation Properties:
                
          
                  
        • animation-name: The name of your @keyframes (e.g., fadeIn).
          • 
                  
        • animation-duration: How long the animation takes (e.g., 2s, 500ms).
          • 
                  
        • animation-timing-function: The speed curve (e.g., linear, ease, ease-in-out).
          • 
                  
        • animation-delay: Wait before starting (e.g., 1s).
          • 
                  
        • animation-iteration-count: How many times to repeat (e.g., 1, 3, infinite).
          • 
                  
        • animation-direction: Play forwards, backwards, or alternate.
          • 
                  
        • animation-fill-mode: Keep the final state after animation (forwards).
                  
          • 
                
        
              
        • 
              
      • 
                More Examples:
                
          
                  
        • 
                    Bounce Animation
                    
          
        @keyframes bounce {
          0%, 100% { transform: translateY(0); }
          50%      { transform: translateY(-40px); }
        }
        .bounce {
          animation: bounce 1s infinite;
        }
                  
          
                    What happens? The element jumps up and down forever.
                  
          • 
                  
        • 
                    Color Change Animation
                    
          
        @keyframes colorChange {
          0%   { background: red; }
          50%  { background: yellow; }
          100% { background: green; }
        }
        .colorful {
          animation: colorChange 3s linear infinite;
        }
                  
          
                    What happens? The background color cycles smoothly through red, yellow, and green.
                  
          • 
                  
        • 
                    Slide In from Left
                    
          
        @keyframes slideIn {
          from { transform: translateX(-100%); }
          to   { transform: translateX(0); }
        }
        .slide-in {
          animation: slideIn 0.8s cubic-bezier(.68,-0.55,.27,1.55) forwards;
        }
                  
          
                  
          • 
                
        
              
        • 
              
      • 
                Advanced: Combining Animations & Multiple Properties
                
          
                  
        • 
                    Multiple Animations
          
                    You can run several animations at once by separating them with commas:
                    
          
        .element {
          animation: fadeIn 1s, moveUp 1s 0.5s;
        }
                  
          
                  
          • 
                  
        • 
                    Animating Multiple Properties
          
                    In your @keyframes, you can animate several CSS properties at the same time:
                    
          
        @keyframes growAndColor {
          from { transform: scale(1); background: blue; }
          to   { transform: scale(1.5); background: orange; }
        }
        .grow {
          animation: growAndColor 1.5s ease-in-out;
        }
                  
          
                  
          • 
                
        
              
        • 
              
      • 
                Tips for Beginners:
                
          
                  
        • Start with simple animations (opacity, color, position).
          • 
                  
        • Use animation-iteration-count: infinite; for looping effects.
          • 
                  
        • Use animation-fill-mode: forwards; to keep the final state after animation.
          • 
                  
        • Experiment with timing functions for different "feels" (e.g., ease,
                    linear, cubic-bezier).
                  
          • 
                  
        • Try combining animations for creative effects!
          • 
                  
          • 
                
        
              
        • 

              
      • 
                Summary:
        
                CSS Animations are a powerful way to add life and interactivity to your website. Start with simple
                effects, then experiment with more complex animations as you get comfortable. Remember: a little
                animation goes a long wayâ€”use it to enhance, not distract!

                What is Image Reflection?
        
                Image reflection is a visual effect where a mirror-like reflection appears below (or above) an image,
                making it look like it's sitting on a shiny surface.
              
        • 

              How to Create Image Reflection in CSS?
        
              You can use the -webkit-box-reflect property (supported in Chrome, Edge, Safari) to add a
              reflection to an image.
          
          
      • 
            Basic Syntax:
            
        
img.reflect {
  -webkit-box-reflect: below 8px linear-gradient(transparent, rgba(0,0,0,0.3));
}
      
        
            
          
              
        • below 8px: Reflection appears below the image, with an 8px gap.
          • 
              
        • linear-gradient: Fades the reflection out for a realistic effect.
          • 
            
        
          
        • 
          
      • 
            Example:
            
        
<img src="photo.jpg" class="reflect" alt="Reflection Example">
      
        
          
        • 
          
      • 
            Tips:
            
          
              
        • This effect works best on images with transparent or solid backgrounds.
          • 
              
        • For more control, adjust the gradient or direction (e.g., above, left,
                right).
              
          • 
              
        • Not supported in Firefox or IE. For cross-browser support, use SVG or a duplicated/flipped image with
                opacity.
